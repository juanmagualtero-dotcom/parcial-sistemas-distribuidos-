# Laboratorio de Comunicación (Cliente-Servidor-UDP)

Este repositorio contiene un proyecto de laboratorio para explorar la comunicación en red usando Python. Incluye ejemplos de un cliente, un servidor y un listener UDP.

## Descripción

*   **`server.py`**: (Añade aquí qué hace tu servidor. Ej: "Servidor TCP básico que recibe mensajes...")
*   **`client.py`**: (Añade aquí qué hace tu cliente. Ej: "Cliente que se conecta al servidor y envía datos...")
*   **`udp_listener.py`**: (Añade aquí qué hace el listener. Ej: "Script que escucha paquetes UDP en un puerto específico...")
